# Como executar o programa
<p>1. Clone o programa para sua máquina escrevendo no terminal <code>git clone link-do-repositorio</code>. Onde "link-do-repositorio" é o link que aparece ao clicar no botão verde "Code" dentro do repositório.</p>
<p>2. Execute em uma IDE (preferencialmente o VS Code) o arquivo <code>main.py</code>.</p>
<p>3. Divirta-se!</p>

# Comandos úteis
<p><code>Ctrl + C</code> --> Copiar figura</p>
<p><code>Ctrl + V</code> --> Colar figura</p>
<p><code>Delete</code> --> Deletar figura</p>
<p><code>Ctrl + F</code> --> Trazer figura para frente</p>
<p><code>Ctrl + T</code> --> Levar figura para trás</p>
<p><code>Ctrl + G</code> --> Agrupar figuras</p>
<p><code>Ctrl + U</code> --> Desagrupar figuras</p>
<p><b>OBS:</b> Ao executar o comando de desagrupar figuras, clique em um espaço em branco para realizar o desagrupamento.</p>

<h3>Como mudar as cores da figura (borda e preenchimento)</h3>
<p>Siga estes passos:</p>
<p>1. No menu superior, na aba ''ferramentas'', selecione a opção 'Selecionar'.</p>
<p>2. Enquanto segura a tecla <code>Shift</code>, clique nas figuras que deseja alterar sua cor, podendo ser uma ou mais figuras.</p>
<p>3. Após selecionar as figuras, solte a tecla <code>Shift</code>.</p>
<p>4. Se quiser trocar a borda, vá para a aba ''Borda'', e escolha dentre as cores disponíveis.</p>
<p>5. Se quiser trocar o preenchimento, vá para a aba ''Preenchimento'', e escolha dentre as cores disponíveis.</p>
